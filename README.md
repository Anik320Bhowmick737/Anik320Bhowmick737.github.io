## 🧠 About

> This is a structured, visual-first approach to learning machine learning, deep learning and reinforcement learning concepts.

Instead of jumping into formulas directly, each topic is built through:
- Intuition first
- Then mathematical formulation
- Then geometric / probabilistic interpretation
- Then interactive visualization

---

## 🎯 What this project focuses on

<div style="background-color:#0d1117; padding:12px; border-radius:10px;">

- Core ML concepts explained visually  
- Step-by-step mathematical buildup  
- Interactive simulations (Plotly / JS)  
- Geometry of ML models  
- Probabilistic interpretation  
- Smooth learning experience for beginners → advanced  

</div>

---

## 🚀 Philosophy

- No blind memorization of formulas  
- Every equation has a reason  
- Every concept is visualized where possible  
- Math is introduced only when intuition is clear  

---

## 🛠️ Tech Stack

- HTML / CSS / JavaScript  
- Plotly.js (interactive plots)  
- GSAP (animations)  
- MathJax (LaTeX rendering)
